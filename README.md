# RodjerCloud — Облачное хранилище в Telegram

Десктопное приложение для использования Telegram как личного облачного хранилища.

## Возможности

- Неограниченное хранилище через ваш Telegram-канал
- Шифрование MTProto
- Drag-and-drop загрузка
- Русский интерфейс
- Авто-синхронизация
- Поиск, теги, избранное
- Тёмная тема

## Сборка

```bash
git clone https://github.com/RodjerYan/RodjerCloud.git
cd RodjerCloud
npm install
npm run build
npm run build:win
```

# RodjerCloud iOS — Полная спецификация

## 1. Навигация

TabBar (внизу) + Stack внутри каждого таба:

- **Главная** (DashboardHome)
- **Мои файлы** (MyFilesPage)
- **Загрузить** (UploadPage)
- **Настройки** (SettingsPage)

Sidebar (десктоп) → на iOS заменён на TabBar + модальное меню для остальных страниц.

## 2. Экран входа (LoginScreen)

**Шаг 1 — Телефон**
- Поле ввода номера телефона
- Кнопка "Отправить код"
- Валидация: непустой номер

**Шаг 2 — Код**
- Поле ввода 4-8 цифр
- Повторная отправка (если код неверный, до 3 попыток)
- Если запрошен 2FA → переход на шаг 3

**Шаг 3 — 2FA**
- Поле ввода пароля (облачный пароль Telegram)
- Кнопка "Подтвердить"

**После успеха:**
- Telegram сессия сохраняется
- Приватный канал "My area" создаётся (если нет — ищется среди диалогов)
- Переход на Dashboard

## 3. Главный экран (DashboardHome)

- Карточки: Всего файлов, Использовано, За неделю, Средний размер
- Круговая диаграмма: файлы по типам (Изображения, Видео, Аудио, Документы, Архивы, Другое)
- Список последних файлов (10 шт.)
- Кнопка быстрой загрузки

## 4. Мои файлы (MyFilesPage) — самый сложный экран

- Переключение: сетка / список
- Поиск по имени
- Сортировка: по имени, размеру, дате
- Фильтр: Все / Изображения / Видео / Аудио / Документы / Архивы / Другое
- Пагинация (50 на страницу)
- Выделение: тап = выбрать, шифт = диапазон
- Массовые действия: скачать, удалить, добавить в избранное, назначить метку
- Превью: изображения показывать в попапе, видео/аудио — проигрывать
- Долгий тап на файле → контекстное меню (скачать, удалить, избранное, метки, заметка, поделиться)

## 5. Загрузка (UploadPage)

- Зона перетаскивания (на iOS — выбор из фотоплёнки / файлов)
- Выбор файлов через системный пикер
- Выбор папки (рекурсивно)
- Очередь с прогресс-баром для каждого файла
- Счётчик: готово, ошибок, всего
- Кнопка "Очистить завершённые"
- Одновременно до 5 загрузок (настраивается)

## 6. Настройки (SettingsPage)

- Внешний вид: уменьшить анимации, компактный режим
- Загрузка: папка для скачивания
- Отправка: авто-переименование, одновременных загрузок (1-5)
- Канал: название, ключ (копирование)
- Кнопка "Сменить канал"
- Опасная зона: очистить данные, сброс настроек

## 7. Остальные страницы

**Статистика:** линейный график (30 дней), круговая по типам, таблица топ-10

**Корзина:** список удалённых, кнопка восстановить, автоочистка через 30 дней

**Избранное:** файлы со звездой, дата добавления

**Общее:** список ссылок, копировать, отозвать, QR-код

**Активность:** лог с фильтром по типу и поиском

**Метки:** создание (имя + цвет), назначение на файлы

**Поиск:** глобальный поиск с фильтрами

**Календарь:** календарь + файлы за выбранный день

**Альбомы:** создание альбомов, добавление файлов

**Заметки:** markdown-редактор для файлов

**Сеть:** пинг до Telegram, график задержки

**Диагностика:** тесты IPC, сессии, хранилища, интернета

- Цвета: фон #0a0a14, текст #e2e8f0, акцент градиент cyan→purple

- Стилизация нативных элементов iOS (navigation bar, tab bar)

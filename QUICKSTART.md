# RodjerCloud — Быстрый старт

## Установка

1. Скачайте `RodjerCloud Setup 1.0.0.exe` из раздела Releases
2. Запустите установщик
3. Введите номер телефона и код подтверждения из Telegram
4. Канал создаётся автоматически

## Использование

- Перетащите файлы в окно загрузки
- Файлы сохраняются в ваш приватный Telegram-канал
- Для восстановления просто войдите в свой аккаунт Telegram
